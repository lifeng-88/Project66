# Vivide

一款风格柔美的 iOS 相册信息查看应用，最低支持 **iOS 15**。

## 功能

- 请求并管理相册访问权限
- 以三列网格浏览相册图片
- **日期筛选**：全部 / 今天 / 本周 / 本月 / 今年 / 自定义区间
- **分类筛选**：收藏 / 截图 / Live Photo / 有位置
- **搜索**：按文件名、分辨率、类型关键词搜索
- **排序**：最新 / 最早 / 分辨率高低 / 文件名 A-Z
- **相册概览**：统计全部、收藏、截图、Live Photo 数量
- **下拉刷新**：重新加载相册
- **分享 / 复制**：详情页一键导出完整图片信息文本
- **批量导入本地**：选择相册图片批量保存到 App 本地存储
- **本地库**：查看、管理已导入图片，支持批量删除与隐藏
- **设置**：多语言（简体中文 / English / 跟随系统）、深色模式（浅色 / 深色 / 跟随系统）
- 查看单张图片的详细信息：
  - 文件名、分辨率、宽高比、文件大小
  - 创建/修改时间、GPS 位置（如有）
  - 媒体类型、来源、收藏/隐藏状态
  - **EXIF 信息**：相机品牌、型号、镜头、焦距、光圈、ISO、快门等
  - Live Photo、HDR、截图、人像景深等特殊标记

## 设计风格

- 玫瑰粉 + 薰衣草紫渐变主题
- 圆角卡片与柔和阴影
- 圆润字体（SF Rounded）
- 定制 App 图标与启动页

## 运行方式

1. 用 Xcode 打开 `vivide.xcodeproj`
2. 选择 iPhone 模拟器或真机
3. 点击 Run（⌘R）
4. 首次运行需授权相册访问

## 项目结构

```
vivide/
├── vivideApp.swift          # 应用入口
├── ContentView.swift        # 根视图与权限路由
├── LaunchScreen.storyboard  # 启动页
├── Theme/AppTheme.swift     # 主题色彩与样式
├── Models/                  # 数据模型
├── Services/EXIFReader.swift
├── ViewModels/              # 相册数据逻辑
└── Views/                   # 界面组件
```

## 技术栈

- SwiftUI
- Photos Framework + ImageIO（EXIF 解析）
- 最低部署目标：iOS 15.0
